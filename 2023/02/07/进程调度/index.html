操作系统期末复习

  1. 进程的组成？

    • pcb程序块（进程存在的唯一标志）
    • 程序段
    • 数据段

  2. 定义

    是计算机中的程序关于某数据集合上的一次运行活动，

    是系统进行资源分配和调度的基本单位。

  3. 进程的状态

    • 创建
    • 就绪
    • 运行
    • 阻塞
    • 结束

  4. 处理机调度

    • 概念：是对处理机进行河里分配，即从就绪队列里按照一定算法选择一个程序并将处理及分配给他运行，以实现进程并发的运行。
    • 方式：剥夺式和非剥夺式

  5. 调度准则

    • cpu利用率

    • 系统吞吐量：单位时间内cpu完成作业是数目

    • 周转时间：作业完成时间-作业的提交时间

      平均周转时间：（完成时间-提交时间)/进程个数

    • 等待时间：进程处于等待时间的和

    • 响应时间：从提交到第一次开始运行的时间

  6. 算法

    • 先来先服务：按照作业到达的先后顺序依次进行调度
    • 短作业优先：作业越短优先级越高
    • 优先级调度算法：优先级高的作业先调度
    • 高响应比优先调度算法：
      • 高响应比算法的优先权：（等待时间+要求服务时间)/要求服务时间=响应时间/要求服务时间
    • 时间片轮转法（公平）：分时系统中，最常见的是基于时间片的轮转调度算法，即让就绪队列上的每个进程每次仅仅运行一个时间片。
    • 多级反馈队列调度算法

  7. 临界区互斥原则（临界区是指在每个进程访问临界资源的那段程序）

    • 空闲让进：如果有若干进程要求进入空闲的临界区，一次仅允许一个进程进入
    • 忙则等待：任何时候，临界区内的进程不可多于一个。
    • 优先等待：进入临界区的进程要在有限时间内推出，以便其他进程能够进入自己的临界区
    • 让权等待：如果进程不能进入自己的临界区，则应让出cpu，避免cpu出现忙等现象。

  8. 死锁

    • 产生原因：非剥夺式的竞争和不恰当的推进顺序
    • 定义：多个资源因竞争资源而造成的一种僵局，如果没有外力，这些进程将无法推进
    • 解决方案：
      • 预防死锁
        • 破环互斥条件
        • 破环不剥夺条件
        • 破环请求和保持条件
        • 破环循环等待条件
      • 避免死锁：安全状态和银行家算法
      • 检测死锁：利用死锁定理
      • 解除死锁：资源剥夺法，撤销进程法，进程回退法

  9. 区别作业调度和进程调度

    • 作业调度：从外存调入内存
      • 进程调度：分配处理机

  10. 信号量：信号量机制是一种有效实现进程同步和互斥的工具

  • 信号量的值大于0表示当前可用资源
  • 信号量小于0表示等待该资源的进程个数
  • 信号量可变，但只能通过pv操作来改变

  1. P操作原语（s）

    • p操作一次，s值减一，即S=S-1（请求分配椅子资源）
    • 如果S>=0，则该进程继续执行，如果S小于0表示无资源，则该进程的状态为阻塞态，把响应的pcb连入该信号队列的末尾，并放弃处理机，进行等待（直至另一个进程执行V（s）操作）
      -临界区是指进程用于访问共享资源的那段代码

  2. V操作原语（s）

    • v操作一次，s值加一。即S=S+1(释放一单位资源)

    • 如果S>0,表示有资源，则该进程继续执行；如果S<=0,则释放信号队列上的第一个pcb所对应的进程（阻塞态改为就绪态），执行V操作的进程继续执行

  3. 银行家算法：

    问系统是否能够安全分配？

    即将可用资源和allocation和需要资源need对比，看是否能够分配出一条安全序列

    • avaliable表示可用资源

      allocation表示已经分配的资源数量

      need表示还需要的资源数量

      work+allocation表示进程结束后释放的资源数量

      work表示当前所剩资源

      如果某个进程提出请求在资源request

      Available = available-request

      need = need - request

      allocatioon = allocation +request

    • process work need allocation Work+allocation finish

  4. 周转时间=结束时间-到达时间

  5. 平均周转时间= （结束时间-到达时间）/作业数量

  6. 对于分页存储管理系统：设页面为P,页面大小为L，d为业内地址=位移量

    P=INT（A/L）:A表示页内逻辑地址

    d = 【A】MOD L
